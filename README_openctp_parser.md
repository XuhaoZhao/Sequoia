# OpenCTP期货合约手续费解析器

## 功能说明

自动解析 http://openctp.cn/fees.html 网页的期货合约手续费数据，并保存为CSV文件。

### 解析字段（共38个）

1. **基础信息**
   - 交易所、合约代码、合约名称、品种代码、品种名称
   - 合约乘数、最小跳动

2. **手续费信息**
   - 开仓费率、开仓费用/手
   - 平仓费率、平仓费用/手
   - 平今费率、平今费用/手

3. **保证金信息**
   - 做多保证金率、做多保证金/手
   - 做空保证金率、做空保证金/手

4. **价格信息**
   - 上日结算价、上日收盘价、最新价
   - 成交量、持仓量

5. **费用计算**
   - 1手开仓费用、1手平仓费用、1手平今费用
   - 做多1手保证金、做空1手保证金、1手市值

6. **盈亏分析**
   - 1Tick平仓盈亏、1Tick平仓净利、2Tick平仓净利
   - 1Tick平仓收益率%、2Tick平仓收益率%
   - 1Tick平今净利、2Tick平今净利
   - 1Tick平今收益率%、2Tick平今收益率%

7. **主力标识**
   - 是否主力合约（通过黄色高亮识别）

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 方式1：立即执行一次

```bash
python parse_openctp_fees.py --once
```

### 方式2：定时执行（每天凌晨1点）

```bash
python parse_openctp_fees.py
```

## 输出文件

1. **CSV数据文件**: `openctp_fees_YYYYMMDD_HHMMSS.csv`
   - 包含所有解析的期货合约数据
   - UTF-8编码，Excel可直接打开

2. **日志文件**: `openctp_parser.log`
   - 记录运行过程和错误信息

3. **控制台输出**:
   - 显示解析的数据总数
   - 显示前10条和后5条数据预览
   - 显示所有字段列表

## 定时任务设置

脚本默认每天凌晨1点自动运行。

如需修改执行时间，编辑 `parse_openctp_fees.py` 第164行：

```python
# 修改为每天凌晨2点
schedule.every().day.at("02:00").do(scheduled_job)

# 或修改为每天下午3点
schedule.every().day.at("15:00").do(scheduled_job)
```

## 注意事项

1. 确保网络连接正常，能访问 http://openctp.cn/fees.html
2. 首次运行建议使用 `--once` 参数测试
3. 日志文件会记录所有运行信息，便于排查问题
4. CSV文件使用时间戳命名，不会覆盖历史数据

## 系统要求

- Python 3.7+
- 网络连接
